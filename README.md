# Project Digital Nao My Dashboard Documentation


## run app in local:

```
npm install
npm run dev
```

## run app in production:

```
npm run build
npm start
```

## run test unit:

```
 npm run test
```

## run test e2e Cypress:

```
 npx cypress open
```
